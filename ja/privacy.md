# プライバシーポリシー

> **日本語版を正本とする**
>
> 本ポリシーの日本語版と他言語版（英語版等）との間に齟齬がある場合は、日本語版を優先します。

## 第 1 条 取得する個人情報

本サービスは、以下の個人情報（個人情報保護法第 2 条第 1 項に定める個人情報のほか、同法第 2 条第 2 項に定める個人関連情報も含みます）を取得・利用します。

### 1.1 AsyncStorage 永続化項目

本サービスは `@react-native-async-storage/async-storage` を使用し、以下の情報を端末内に保存します。これらはサーバーに送信されません。

| 保存キー            | 内容                                 | 目的                                       |
| ------------------- | ------------------------------------ | ------------------------------------------ |
| `hasSeenIntro`      | オンボーディング完了フラグ           | 初回起動時のチュートリアル表示制御         |
| `consentState`      | 規約同意状態                         | 利用規約・プライバシーポリシーへの同意管理 |
| `subscriptionStore` | 購読状態・次回更新日・trial 終了日   | 課金状態に応じた機能ゲーティング           |
| `settingsStore`     | 音声設定・テーマ設定・セッション設定 | ユーザー設定の跨セッション復元             |
| `customDecksStore`  | ユーザー作成デッキの単語リスト・名前 | マイデッキ機能の永続化                     |

### 1.2 RevenueCat 経由の購入情報

本サービスは RevenueCat SDK を使用し、以下の情報を RevenueCat サーバー上で管理します。運営者側サーバーには保存しません。

- 匿名 customer ID（RevenueCat 自動発行）
- 購入履歴・購読状態・trial 利用状況
- 請求元（Apple / Google）の識別子

### 1.3 Expo SDK が収集する技術情報

本サービスは Expo SDK の一部機能を使用し、以下の技術情報を収集します。

- `expo-device`: デバイスモデル名・OS バージョン・OS ビルド ID
- `expo-updates`: OTA アップデートのリクエストログ（更新日時・バージョン）
- `expo-constants`: アプリバージョン・ビルド番号・Bundle ID / Package name

### 1.4 アプリ内で生成・利用されるが保存しないもの

以下の情報は、セッション中または一時的に利用しますが、端末内・サーバーともに永続化しません。

- セッション中の音声再生ログ（再生中のみメモリ上に保持し、セッション終了時に破棄）
- 解約理由の自由記述（解約 CTA 押下と同時に破棄）

### 1.5 取得しないもの（明示的事項）

本サービスは、以下の情報を取得しません。

- 位置情報（GPS 等）
- マイク・カメラ（`expo-audio` は音声再生のみで、録音権限は無効化しています）
- 連絡先・写真ライブラリ
- IDFA / 広告 ID（`expo-tracking-transparency` は未導入、広告 SDK は未導入）
- 生体情報・HealthKit データ
- ユーザーの氏名・メールアドレス等、個人を特定できる情報（Apple ID / Google Play アカウント情報は本サービス側では取得しません）

## 第 2 条 利用目的

取得した個人情報は、以下の目的の範囲内で利用します。目的の範囲を超えた利用は行いません。

1. **サービス提供・維持**: 購読状態に応じた機能の有効化・無効化、サブスクリプション管理、無料体験期間の管理
2. **ユーザー設定の永続化**: 音声設定・テーマ設定・セッション設定・マイデッキ等、跨セッションでの設定復元
3. **カスタマーサポート**: 購入復元トラブル対応のため、RevenueCat customer ID を照会・確認
4. **アプリ改善**: OS バージョン別の動作確認、OTA アップデートの配信状況確認（なお、Crashlytics 等のクラッシュ分析ツールは未導入のため、当面は EAS Update の基本的なログのみ）
5. **規約同意の管理**: 利用規約・プライバシーポリシー改訂時の再同意制御

## 第 3 条 第三者提供 / 共同利用

1. 法令に基づく開示を除き、ユーザー個別の同意なしに、個人情報を第三者に提供することはありません。
2. 個人を特定できない形式に加工した統計情報については、アプリ改善の目的で利用・公表することがあります。

本サービスは、以下の事業者に業務の一部を委託しており、必要な範囲で個人情報の取り扱いを委託しています。

## 第 5 条 越境移転

1. 前条の委託先の一部は、日本国外に所在するため、取得した個人情報が日本国外へ移転される場合があります。
2. 越境移転にあたっては、標準的契約条項（Standard Contractual Clauses）等の適切な保護措置の下で行います。
3. 個人開発者運営のため、大規模な ISMS 認証は取得していませんが、通信は HTTPS で暗号化され、委託先各社のプライバシーポリシー・セキュリティ基準に準拠しています。

## 第 6 条 安全管理措置

1. 個人情報の漏えい・滅失・毀損の防止のため、以下の安全管理措置を講じます。
   - 端末内の AsyncStorage に保存し、機微情報（パスワード・認証トークン・決済情報等）は保存しません
   - 決済関連情報は App Store / Google Play / RevenueCat が管理し、当アプリ側では保持しません
   - 通信は HTTPS（公開 URL / RevenueCat API / Expo Updates）を使用します
2. 本サービスは個人開発者による運営であり、大規模な情報セキュリティマネジメントシステム（ISMS）の認証は未取得です。

## 第 7 条 開示・訂正・利用停止請求の方法

1. ユーザーは、保有個人データの開示、訂正、利用停止等を請求することができます。
2. 請求は、第 13 条の連絡先メールアドレス宛てに行ってください。本人確認のため、RevenueCat customer ID 等の情報をお知らせいただく場合があります。
3. 請求を受け付けた場合、原則として 14 営業日以内に回答いたします。
4. AsyncStorage 内のデータについては、アプリ内「設定」>「すべてのデータを削除」またはアプリのアンインストールにより、端末上から消去できます。

## 第 8 条 Cookie / トラッキング

1. 本サービスはネイティブアプリであり、Cookie は使用しません。
2. 広告 SDK・トラッキング SDK は導入していません。
3. 分析ツール（Google Analytics / Firebase Analytics 等）は導入していません。

## 第 9 条 16 歳未満の取り扱い

1. 16 歳未満のユーザーは、保護者の同意を得た上で本サービスを利用してください。
2. 本サービスは、年齢確認を行う UI を設けておりません（ソフト同意モデル）。保護者の監督の下でご利用ください。

## 第 10 条 漏えい時通知

1. 個人情報の漏えい・滅失・毀損が発生した場合、個人情報保護委員会への報告および、影響を受ける可能性のあるユーザーへの通知を行います。
2. 通知方法は、本サービス内のお知らせ、またはメール等の適切な手段を用います。

## 第 11 条 改訂

1. 本ポリシーは、法令の改訂やサービス内容の変更等に応じて改訂されることがあります。
2. 改訂内容は、本サービス内の通知および公開 URL において公表します。
3. 重要な改訂（取得項目の追加、第三者提供範囲の変更等）については、`legalManifest.json` の `minRequiredRevision` を更新し、ユーザーの再同意を求める場合があります。
