# 特定商取引法に基づく表示

特定商取引法に基づき、以下の事項を表示いたします。

| 項目                 | 内容                                                                                                                                                                                                                                                                |
| -------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **販売事業者**       | <個人事業者の代表者氏名 / 商号>                                                                                                                                                                                                                                     |
| **運営統括責任者**   | 上記販売事業者と同じ個人                                                                                                                                                                                                                                            |
| **所在地**           | <個人事業者の代表住所>                                                                                                                                                                                                                                              |
| **電話番号**         | <個人事業者の連絡先電話番号>                                                                                                                                                                                                                                        |
| **メールアドレス**   | <個人事業者の連絡先メールアドレス>                                                                                                                                                                                                                                  |
| **販売価格**         | 月額 240 円（税込）                                                                                                                                                                                                                                                 |
| **支払方法**         | Apple ID（App Store）/ Google Play アカウント                                                                                                                                                                                                                       |
| **商品引渡時期**     | 購入確認後、即時にアプリ内の Premium 機能が有効化されます                                                                                                                                                                                                           |
| **解約方法**         | **iOS の場合**: iPhone の「設定」アプリ > ご自身の名前（Apple ID）> 「サブスクリプション」>「MindDrift Premium」>「解約する」<br>**Android の場合**: Google Play アプリ > プロフィールアイコン >「定期購入とサブスクリプション」>「MindDrift Premium」>「解約する」 |
| **動作環境**         | **iOS**: iOS 15.2 以降<br>**Android**: Android 7.0（API 24）以降                                                                                                                                                                                                    |
| **返品・キャンセル** | デジタルコンテンツのため、購入後の返品・返金は原則としてお受けできません。解約は次回更新日まで有効です。返金に関する特例は、Apple Inc. または Google LLC のポリシーに準じます                                                                                       |

## 申し込み確定に関する表示

Premium サブスクリプションへのお申し込みにあたり、申し込み確定の CTA ボタンには「**240 円/月で申し込む**」等の文言を表示します。「次へ」等の曖昧な表現は使用しません。

## 電話番号の表示について

特商法施行規則第 23 条に基づき、電話番号の常時表示を推奨いたします。個人プライバシー保護の観点から、表示を控える場合は「請求があった場合、遅滞なく開示いたします」の対応も許容されます。
