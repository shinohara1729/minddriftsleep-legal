# 特定商取引法に基づく表示

特定商取引法に基づき、以下の事項を表示いたします。

| 項目                 | 内容                                                                                                                                                                                                                                                                |
| -------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **販売事業者**       | <個人事業者の代表者氏名 / 商号>                                                                                                                                                                          |
| **運営統括責任者**   | 上記販売事業者と同じ個人                                                                                                                                                                                 |
| **所在地**           | <個人事業者の代表住所>                                                                                                                                                                                   |
| **電話番号**         | 請求があった場合に遅滞なく開示します（特定商取引法第 11 条ただし書、同法施行規則第 25 条）。請求は下記メールアドレス宛にご連絡ください                                                                   |
| **メールアドレス**   | <個人事業者の連絡先メールアドレス>                                                                                                                                                                       |
| **販売価格**         | 月額 240 円（税込）                                                                                                                                                                                      |
| **支払方法**         | Apple ID（App Store）/ Google Play アカウント                                                                                                                                                            |
| **商品引渡時期**     | 購入確認後、即時にアプリ内の Premium 機能が有効化されます                                                                                                                                                |
| **解約方法**         | お買い上げ後のサブスクリプションは、Apple ID（App Store）または Google Play アカウントの定期購入管理画面からいつでも解約できます。詳細な操作手順は各ストアのヘルプをご参照ください。解約後も次回更新日までは引き続き Premium 機能をご利用いただけます |
| **動作環境**         | **iOS**: iOS 18 以降を推奨<br>**Android**: Android 13 以降を推奨                                                                                                                                         |
| **返品・キャンセル** | デジタルコンテンツの性質上、購入後の返金は原則として行いません。解約をご希望の場合は前項「解約方法」をご参照ください                                                                                     |

## 申し込み確定に関する表示

Premium サブスクリプションへのお申し込みにあたり、申し込み確定の CTA ボタンには「**240 円/月で申し込む**」等の文言を表示します。「次へ」等の曖昧な表現は使用しません。
